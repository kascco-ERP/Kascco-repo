Overview

you can create your approval in any modules without any custamizations. The app will support mult state and multi approvers.

Configuration

Approver creating module need to have Header, State field with selection Type, Company Field and Context. The Approval is only working with after module configration records. After the app installation you set the group in General Settings -> Users

Features

  • Support Multi Approvers in sequence wise and parallel.
  • Support MUlti States.
  • Support Multi Company.
  • Rejected Record Resend Option.
  • Approvers Updating Option.
  • Email Notifications for Approver and Requester.
  • Status Adding in Tree view .
  • Disable Report Print Button.
  • Approval History In Chatter.
